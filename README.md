## How to install 

```bash
  # Clone the project
  git clone https://github.com/whosramoss/react-blackboard/

  # Go to the project directory
  cd react-blackboard

  # Install dependencies
  npm install

  # Start the server 
  npm run dev
```

## How to build 

```bash
  npm run build
```
